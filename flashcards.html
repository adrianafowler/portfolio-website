Flash Cards

Vocabulary application

Flash Cards is a project of an application
for users wishing to learn another language and is intended to help in memorising new vocabulary.

01 Research

Competitive analysis:

Analysis was conducted on three competitors:
Quizlet, Simple Flashcards and English Vocabulary.
To assess usability I took into consideration the following factors:
Launch experience, navigation and task completion difficulty, functionality, buttons and interface design and settings.
There are two main functions which flash card app should provide: an easy way to create card set and revision methods.

Quizlet provides users with very easy, clean and intuitive navigation around two sided cards. It gives the user the opportunity to revise through tests.
Introduction to product functions is minimalistic and clear.
Simple Flashcards provides users with more services which are free of charge, for instance:
offline use and adding images. Additionally it has written text recognition function and the app is compatible with Quizlet.
While analysing English Vocabulary app, I identified some functionality issues: cards were only one sided with no possibility to customise.

Interviews:

I interviewed five people who were learning a foreign language. Each participant answered five questions about using an application as a tool. I researched how they memorise new vocabulary, which apps they use and their expectations from a flash card app.

Questions:

"Are you currently learning a foreign language or did you learn any in the past?"
"Could you describe how you learn/memorise new vocabulary, what technics you use?"
"What is the biggest challenge in learning a new language?"
"Have you ever used an application for learning new vocabulary? If not, would you? Which one?"
"Which application do you have on your mobile?"

Findings:

Visual representation of the word is the most important in learning process.
Making notes while learning is very helpful.
Participants mentioned several times games and quizzes to be a great tool to support learning process.
One of the biggest challenges in learning a new language is pronunciation and remembering new words.
Only one participant admitted to be a user of language learning app, whereas the others expressed their interest in trying one, only if it had following features:
quizzes and games, progress check, audio function for words (pronunciation), possibility of making notes and setting reminders.

Task analysis:

Having a hypothetical solution for my persona, the next step was to create one which is practical. I’ve started with user flows and sketches and then I designed digital wireframes:

User Task 1:

Goal: Create a new flash card
Entry point: Tap on “ create a card” on the home screen
Success criteria: Have a card set which will be useful tool for learning new vocabulary.

User Task 2:

Goal: Take a quiz
Entry point: Tap on ‘quiz’ on the home screen
Success criteria: User practiced a set of words
